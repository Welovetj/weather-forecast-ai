# Cloudora Weather App (PM Accelerator Technical Assessment)

   Demo video: https://drive.google.com/file/d/1GOr_XV-uhuQ4KkQN9aJssxX3-ZpSPdLg/view?usp=sharing

This repository contains my PM Accelerator technical assessment submission.

The app code is in the `Skye/` folder and is built with React Native + Expo.

## What Is In This Repo

- Mobile weather app with city search and current conditions
- Forecast display and weather details
- Geolocation-based weather lookup
- Local search history (SQLite) with edit and delete
- History export to CSV and JSON
- Optional Supabase sync for saved history
- Favorites and city comparison support
- Theme toggle and temperature unit toggle

## Project Location

- Main app: `Skye/`
- App documentation: `Skye/README.md`
- Package requirements list: `Skye/requirements.txt`

## Quick Start

1. Open a terminal in `Skye/`
2. Install dependencies:

   npm install

3. Copy environment template and fill your values:

   copy .env.example .env

4. Start app:

   npx expo start --clear
